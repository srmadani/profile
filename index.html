Projects

A selection of research and development projects.

Targeted Demand Response Through Critical Peak Pricing

Developed reinforcement learning models (PPOD, SACD, D3QN) to analyze the impact of Critical Peak Pricing (CPP) on peak load shaving in electricity grids, considering prosumer behavior with DERs (PV, batteries, EVs). Proposed targeted dynamic pricing strategies.

View on GitHub

Investment in Vehicle-to-Grid and Distributed Energy Resources

Performed cost-benefit analysis from distributor and prosumer perspectives for investments in DER, V2G, and EMS. Analyzed the impact of different rate structures using optimization models (Gurobi) and real-world data from Vermont homes.

View on GitHub

Reinforcement Learning for Demand Response in Commercial Buildings

Created an RL framework to optimize HVAC (discrete) and lighting (continuous) systems in commercial buildings using algorithms like SACD, PPOD, TD3, PPO. Aimed to reduce costs, CO2 emissions, and peak load while maintaining comfort, incorporating risk metrics (VaR, CVaR).

View on GitHub

Financial Q&A RAG System

Built a Retrieval-Augmented Generation (RAG) system using ElasticSearch and a local LLM (Ollama/Qwen) to answer financial questions based on 10-K filings dataset. Provided a Streamlit interface and dockerized deployment.

View on GitHub

EV Smart Charging Telegram Bot

Developed a Telegram bot using an LSTM model for forecasting electricity prices/emissions and the Gemini LLM for natural language understanding. Recommends optimal EV charging schedules to minimize cost and environmental impact based on user inputs.
